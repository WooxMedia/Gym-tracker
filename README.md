# Gym-tracker — Carnet de Gym

Carnet d'entraînement mobile (PWA) : split 4 jours (Push / Pull / Legs+Abs / Upper),
saisie charge (lbs) + répétitions, timer de repos automatique, chrono de séance,
historique avec volume total. Données stockées localement dans le navigateur
(`localStorage`) — rien n'est envoyé sur un serveur.

**App : https://wooxmedia.github.io/Gym-tracker/**

Sur Android : ouvrir le lien dans Chrome → menu ⋮ → « Installer l'application »
(ou « Ajouter à l'écran d'accueil »). Fonctionne ensuite hors-ligne.

Développé dans `WooxMedia/WooxClaude` (`apps/gym`), déployé ici via GitHub Pages
(workflow `.github/workflows/pages.yml`).
